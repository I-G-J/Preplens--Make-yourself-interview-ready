# 🚀 PrepLens — AI Interview Question Predictor

PrepLens is an AI-powered web application that helps students and developers prepare for interviews smartly by analyzing Job Descriptions (JDs) and generating relevant interview questions.

---

## 🧠 Problem

Most candidates prepare randomly for interviews without knowing:
- What skills actually matter for a specific job
- What kind of questions will be asked

This leads to inefficient and unfocused preparation.

---

## 💡 Solution

PrepLens solves this by:
- Extracting key skills from any Job Description
- Prioritizing them intelligently
- Generating structured interview questions

---

## ⚙️ Features

✅ Paste any Job Description  
✅ Smart skill extraction  
✅ Multi-layer skill prioritization  
✅ Question generation (Easy → Medium → Hard)  
✅ Clean and responsive UI

**AI Integration:**
- Hugging Face API

---

## 🔄 How It Works

1. User pastes a Job Description
2. Backend processes and cleans the text
3. Extracts important skills using logic + AI
4. Prioritizes skills using:
   - Frequency analysis
   - Position-based weighting
5. Generates structured interview questions
6. Sends results to frontend
